# NFL Draft Prediction

## 概要
NFLの選手データをもとに、ドラフトで指名されるかどうかを予測する2値分類タスク。

- **タスク**: ドラフト指名の有無を予測（2値分類）
- **データ**: 身体能力テスト・ポジション情報など
- **結果**: AUC 0.80792（ベースライン）→ 0.84705
- **順位**: 107位 / 3434人中（上位3.1%）

## 主な工夫
- EDAによるデータの可視化と特徴量エンジニアリングの方針策定
- ドメイン知識（ASI・ポジション別特性）の調査と特徴量への反映
- BMI・ASI構成要素・Age_missingフラグなどの特徴量エンジニアリング
- Random ForestからXGBoostへのモデル変更
- Optunaによるハイパーパラメータチューニング
- K-Fold交差検証 + Early Stoppingによる過学習抑制
- 特徴重要度のCV分析による特徴量の評価

## 使用技術
- Python / XGBoost / Optuna / pandas / matplotlib
