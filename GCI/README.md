# Prediction

## 概要
スポーツ予測に関する2値分類タスク

- **タスク**: 選手データからドラフト指名の有無を予測（2値分類）
- **データ**: 構造化データ
- **結果**: AUC0.84705
- **順位**: 上位3.1%

## 主な工夫
- EDAによるデータの可視化と特徴量エンジニアリングの方針策定
- ドメイン知識の調査と特徴量への反映
- XGBoostへのモデル変更
- Optunaによるハイパーパラメータチューニング
- K-Fold交差検証 + Early Stoppingによる過学習抑制
- 特徴重要度のCV分析による特徴量の評価

## 使用技術
- Python / XGBoost / Optuna / pandas / matplotlib
