# ドラフト指名予測（スポーツ分析・2値分類）

## 概要
選手データからドラフト指名の有無を予測する2値分類タスク。
テーブルデータに対してEDA・特徴量エンジニアリング・
モデル選定を一貫して取り組んだ。

- **結果**: AUC 0.84705
- **順位**: 上位3.1%

## 技術的な工夫と意思決定

### EDAとドメイン知識の活用
特徴量の分布を可視化した際、ある特徴量が二峰性の分布を
示していた。ドメイン知識の調査によりその特性が2分される
理由を特定し、特徴量エンジニアリングの方針策定に活用した。
また、可視化により欠損値が多い特徴量を見つけ、意味を考える
きっかけになった。

### 特徴量エンジニアリング
効果があった工夫：
- 複数の特徴量から一般的な指標を新規生成（AUC向上に寄与）
- 欠損値を「情報」として扱い、欠損フラグを明示的に追加。
  特定の特徴量が欠損している選手には後ろ向きな理由がある
  という仮説を立て、それが分類精度に影響すると判断した。
- XGBoostは欠損値をそのまま扱えるため、
  欠損値の補完は行わない方が精度が高かった。

### モデル選定
ベースラインのRandom ForestからXGBoostに変更。
逐次的に誤りを修正する勾配ブースティングの方が
本タスクに有効と判断した。

### 過学習抑制と評価
- K-Fold交差検証 + Early Stoppingで過学習を抑制
- 特徴重要度のCV（変動係数）を各Fold間で計算し、
  ばらつきが小さく重要度が高い特徴量を優先的に採用
